# Cursor Rules Overview

This directory contains custom rules (`.mdc` files) that govern the behavior of the Cursor AI for the SoloSuccess AI project. These rules ensure consistency across code generation, design patterns, and adherence to production-grade standards.

## Project Standards

- [00-project-standards.mdc](00-project-standards.mdc): General coding principles and production-ready requirements.
- [10-components.mdc](10-components.mdc): Standards for React components and UI development.
- [20-apis.mdc](20-apis.mdc): Rules for API route design and implementation.
- [30-accessibility.mdc](30-accessibility.mdc): Accessibility (a11y) mandates and best practices.
- [40-logging.mdc](40-logging.mdc): Logging and debugging guidelines.
- [50-nextjs-static-generation.mdc](50-nextjs-static-generation.mdc): Guidelines for Next.js static and dynamic generation.

## How to use

Cursor automatically detects these rules. You can also reference them explicitly by typing `@` and selecting the rule.
